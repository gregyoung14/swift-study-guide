## üöÄ Getting Started

### Local Development

1. **Clone the repository:**
   ```bash
   git clone https://github.com/gregyoung14/swift-study-guide.git
   cd swift-study-guide
   ```

2. **Install dependencies:**
   ```bash
   pip install -r requirements.txt
   ```

3. **Run the development server:**
   ```bash
   mkdocs serve
   ```

4. **Open your browser:**
   Navigate to `http://127.0.0.1:8000/`

### Deployment

This project is configured to deploy on Vercel. Simply connect your GitHub repository to Vercel, and it will automatically build and deploy on every push to the `master` branch.

**Live Site:** https://swift-study-guide.vercel.app
